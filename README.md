## Dataset
Netflix Movies and TV Shows Dataset
The dataset contains information about titles, genres, popularity scores, 
release years, and other content-related attributes used for analysis.

## Key Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Genre Analysis
- Popularity Analysis
- Release Year Analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Drama was the most frequent genre in the dataset.
- Drama was also the most popular genre among highly voted content.
- Spider-Man: No Way Home had the highest popularity score.
- The United States, Thread had the lowest popularity score.
- 2020 recorded the highest number of movie releases.

## Business Recommendations
- Continue investing in Drama content due to its strong representation and popularity.
- Analyze successful Action and Adventure titles to identify audience preferences.
- Study low-performing content to improve future acquisition decisions.
- Use genre popularity insights to guide content recommendations.
